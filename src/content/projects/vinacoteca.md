## Descripción del proyecto

He desarrollado esta landing page utilizando **React**, enfocada en presentar una **vinacoteca online** con una selección de cervezas artesanas y vinos cuidadosamente elegidos.  
El objetivo del proyecto es ofrecer una interfaz rápida, moderna y fácil de navegar, conectada a una API propia para gestionar el catálogo de productos.

## ¿Qué hace?

- Catálogo dinámico de **cervezas** y **vinos** con imágenes, descripciones y precios  
- Integración con API REST desarrollada en **Node.js** y almacenada en **MongoDB**  
- Filtros por tipo de bebida, origen, variedad y graduación  
- Navegación fluida gracias a **React Router**  
- Diseño responsive con **Tailwind CSS**  
- Gestión del estado y consumo de API con hooks personalizados  
- Despliegue optimizado en **Vercel**

## Decisiones técnicas

Elegí **React** por su flexibilidad, su ecosistema maduro y la facilidad para crear componentes reutilizables.  
La API está construida con **Node.js** y **MongoDB**, lo que permite manejar el inventario de forma escalable y mantener una estructura de datos flexible.

El diseño está implementado con **Tailwind CSS**, lo que facilita mantener una UI coherente, modular y altamente personalizable sin necesidad de escribir CSS repetitivo.
